How it lands in Raycast.

Raycast distributes themes as raycast://theme?… deeplinks — there's no file format. A link is the whole theme.

Click Install in Raycast above. Raycast opens with a preview of the palette applied — confirm, and it's active.

scope

What Raycast lets you theme. And what it doesn't.

Unlike VS Code, Raycast's theming surface is small — but the handful of things it exposes cover most of the visual identity.

Themeable

Exactly 12 colors plus appearance (light/dark): 5 UI slots — background, secondary background, text, selection, loader — and 7 semantic accents: red, orange, yellow, green, blue, purple, magenta. That's the entire shareable surface.

Not themeable

Typography. Corner radius. Window background mode (solid vs gradient is a local Raycast setting, not part of a shared theme). Row height and density. Icon shapes. Command-list layout. The “⌥ Space” summon hotkey. Individual extension icons.

Per-user, not themeable

Which commands appear in the root view, keyboard shortcuts, AI model selection. These are user settings, not part of a distributable theme.
