# Supabase Theme Kits

A collection of Supabase-themed customization kits for your desktop tools — including Chrome, VS Code, Raycast, terminal, and wallpapers — built in admiration of the Supabase design system.

> Just a fun side project. Don't read too much into it.

## Usage

```bash
git clone https://github.com/colings86-work/supabase-theme-kits.git
cd supabase-theme-kits
open index.html
```
