installation

Pick your terminal. We wrote the steps.

Each format is a plain file you can drop in. No sign-in, no package manager, nothing to run.

iTerm2

Imports directly into iTerm's color presets panel. Takes under a minute.

  1. Unzip the bundle and locate the .itermcolors file.
  2. Open iTerm2 and go to Settings → Profiles → Colors.
  3. Click Color Presets… → Import… at the bottom-right of the colors panel.
  4. Pick the .itermcolors file. It's now in the dropdown.
  5. Click Color Presets… again and select your new theme to apply.

macOS Terminal.app

The built-in Terminal works fine with our ANSI colors — just slightly more clunky to import.

  1. Unzip the bundle and locate the .terminal file.
  2. Double-click the file — Terminal opens a new window using that profile.
  3. Open Terminal → Settings → Profiles and find the new profile in the list.
  4. Click Default at the bottom to make it the default for new windows.

Kitty

Include the color config from your main kitty.conf.

  1. Copy the .conf file into ~/.config/kitty/.
  2. Add a line to your main kitty.conf: include <filename>.conf.
  3. Reload config with Ctrl+Shift+F5, or restart Kitty.

Alacritty

The TOML fragment is just the colors section of your config.

  1. Open your Alacritty config (~/.config/alacritty/alacritty.toml).
  2. Either paste the contents of our .toml file in, replacing any existing [colors.*] sections, or import it with import = ["<path>.toml"] at the top.
  3. Save. Alacritty hot-reloads on config changes.

Ghostty

Either include it as a theme file or paste into your main config.

  1. Copy <name>-ghostty.conf into ~/.config/ghostty/themes/ (create it if missing).
  2. In your main ~/.config/ghostty/config, add theme = <name>-ghostty.
  3. Reload Ghostty (Cmd+Shift+, on macOS) or restart.

Warp

Warp looks for user themes in its themes folder.

  1. Copy the .yaml file into ~/.warp/themes/ (create it if missing).
  2. In Warp, open Settings → Appearance → Themes.
  3. Find the new theme under Custom Themes and select it.
