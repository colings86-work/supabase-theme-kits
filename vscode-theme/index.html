installation

Pick your editor. We wrote the steps.

The same .vsix works across VS Code, Cursor, Windsurf and VSCodium. For the JSON-only route, paste the settings snippet straight into settings.json.

VS Code

Install the .vsix directly — no publishing, no sign-in.

  1. Unzip the bundle and locate the .vsix file.
  2. Open VS Code and press Cmd+Shift+P (or Ctrl+Shift+P). Run Extensions: Install from VSIX… and pick the file.
  3. Alternative: run code --install-extension <file>.vsix in your terminal.
  4. Open the command palette again and run Preferences: Color Theme. Pick your Supabase theme from the list.

Cursor

Cursor uses the VS Code extension format — the same .vsix installs cleanly.

  1. Open Cursor and press Cmd+Shift+P (or Ctrl+Shift+P).
  2. Run Extensions: Install from VSIX… and select the downloaded file.
  3. Run Preferences: Color Theme and pick your Supabase theme.

Windsurf

Windsurf accepts VS Code extensions through the same VSIX flow.

  1. Open Windsurf's command palette (Cmd+Shift+P).
  2. Run Extensions: Install from VSIX… and pick the file.
  3. Switch the theme via Preferences: Color Theme.

VSCodium

The same VSIX works — VSCodium shares VS Code's extension format.

  1. Open VSCodium and press Cmd+Shift+P.
  2. Run Extensions: Install from VSIX… and pick the file.
  3. Alternative CLI: codium --install-extension <file>.vsix.
  4. Run Preferences: Color Theme to activate.

settings.json snippet

If you don't want to install anything, paste the settings-snippet.json contents into your settings.json under workbench.colorCustomizations and editor.tokenColorCustomizations. Changes apply live.

  1. Open settings as JSON: palette → Preferences: Open User Settings (JSON).
  2. Paste the contents of settings-snippet.json at the top level of the object, merging if you already have those keys.
  3. Save. The theme applies immediately — no restart required.
