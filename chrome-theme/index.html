installation

Two minutes, no sign-in. Just drop it in.

These are unpacked extensions, perfect for work laptops where you can't hit the Web Store. Download, unzip, load.

1Build & download

  1. Open the Theme Builder above and tweak the chrome to taste.
  2. Hit download theme folder — you get a zip with manifest.json, images, and icons.
  3. Unzip somewhere stable — not Downloads. ~/chrome-themes/supabase-custom is a good home.

2Load it into Chrome

  1. Visit chrome://extensions.
  2. Toggle Developer mode on (top-right).
  3. Click Load unpacked and select the unzipped folder.
  4. Chrome applies the theme instantly. Reset from Settings → Appearance → Reset to default.
