A weekend project, greener than most.

Anthropic shipped Claude Design just as I was about to start at Supabase. I was excited to try the new product on something real, not just a throwaway prompt.

It was the weekend before my start date, which seemed like serendipitous timing. Theming my new work laptop in Supabase green felt like the most fitting pre-start project I could think of — a small tribute to somewhere I'd admired from afar.

Wallpapers first, then the browser, then the editor, then the terminal, then the launcher. One kit at a time, each one drawing from Supabase's public design system, each one small enough to finish in an evening. This theme-building site unexpectedly popped out along the way — unofficial and fan-made, a love letter from someone who isn't a designer.

The result

Five kits shipped over a weekend — wallpapers, browser, editor, terminal, launcher — each builder tweakable and downloadable.

What's in the set

Five live builders. Each one is its own page with knobs, a live preview and a download. None of it is affiliated with Supabase — it's all fan-made from their public design system.

01 Wallpapers Nine shapes, any resolution, SVG + PNG + ZIP. → 02 Chrome theme Three near-black variants. Loads unpacked. → 03 VS Code theme Five palettes. Ships as a .vsix for any VS Code fork. → 04 Terminal theme Five palettes for iTerm, Kitty, Alacritty, Ghostty, Warp. → 05 Raycast theme Six dark + one light. One-click raycast:// install. →

How it's made

Every kit here was co-designed with Claude Design. That was the whole point — the new product dropped, and building something I'd actually use felt like the right way to try it out.

The design tokens come from Supabase's public design system. The visual vocabulary — #3ECF8E, the two-tone wordmark, Source Code Pro for mono, near-black backgrounds — is theirs; the kits are just new surfaces for it.

Each builder is a static HTML page with live previews and in-browser exporters. No build step, no server, no tracking.

Thanks

To the Supabase design team for making a system so consistent it's fun to theme things around. To Anthropic for shipping Claude design at the perfect time, and making it a product that a non-designer could use to build this in just a weekend. Everything here is unofficial.
